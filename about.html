Project Information

This website was created for IT 3203 at Kennesaw State University. The purpose of this project is to explain the evolution of web servers and how server technologies continue to support the modern Internet.

About the Author

Name: Nathan Wymer

Major: Information Technology

University: Kennesaw State University

Project Goals

  • Explain the history and development of web servers.
  • Research modern technologies such as cloud computing and HTTP/3.
  • Create a responsive website that works on desktop and mobile devices.
  • Apply HTML, CSS, and JavaScript concepts learned in IT 3203.

Responsive Design Implementation

The website was designed using responsive web design principles. CSS media queries allow the layout, navigation menu, images, tables, and quiz elements to adjust based on screen size.

The goal was to create a website that provides a consistent experience for desktop users, tablet users, and smartphone users.
