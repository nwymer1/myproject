Basic Terms

  • HTTP — Protocol for requesting web pages and files.
  • HTTPS — Secure HTTP using TLS encryption.
  • DNS — Resolves domain names to IP addresses.
  • Load balancing — Distributes traffic across servers.
  • Caching — Stores copies of content to speed delivery.
  • Reverse proxy — Front-end server that manages traffic to backend servers.
  • Serverless — Run code without managing servers; scales automatically.

Static vs Dynamic

Static files are served as-is; dynamic content is generated by server-side code (Node.js, PHP, Python, etc.).

Additional Concepts

  • SSL/TLS — Encrypts communication.
  • CDN — Distributes content globally.
  • HTTP/3 — Uses QUIC for improved performance.
  • Containers — Lightweight packaging for consistent deployment.
  • Virtual Machines — Full OS instances for hosting servers.
