Milestones

CERN HTTPd (1990)

Created by Tim Berners-Lee; introduced the HTTP request/response model and hosted the first website.

NCSA HTTPd (1993)

Added CGI support and easier configuration, enabling dynamic content generation.

Apache HTTP Server (1995)

Open-source server that became widely used because of its modular design and community support.

Nginx (2004)

Designed for high concurrency with an event-driven architecture to handle many simultaneous connections.
