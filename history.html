CERN HTTPd (1990)

The first web server was created by Tim Berners-Lee at CERN, and it introduced the HTTP request/response model and hosted the first website. This single-host model established the client/server pattern that underpins the web today.

NCSA HTTPd (1993)

NCSA HTTPd added CGI support and easier configuration which enabled dynamic content generation and wider adoption. Its code and ideas were the foundation for later servers.

Apache HTTP Server (1995)

Apache grew from patches to NCSA HTTPd and became the dominant open-source server in the late 1990s due to its modular architecture, community support, and cross platform stability.

Nginx and the C10K problem (2004)

Nginx was designed to handle many simultaneous connections with an event driven architecture. This made it ideal for high concurrency and low memory environments.

Timeline

Why these milestones matter

Each milestone introduced a core capability: the HTTP model (CERN), dynamic content (NCSA), extensibility and community development (Apache), and high concurrency and event driven design (Nginx). These shaped how servers are built and deployed today.
