Milestones

CERN HTTPd (1990)

The first web server created by Tim Berners-Lee introduced the HTTP request/response model and hosted the first website.

NCSA HTTPd (1993)

NCSA HTTPd added CGI support and easier configuration enabling dynamic content generation and wider adoption.

Apache HTTP Server (1995)

Apache grew from community patches and became the dominant open-source server due to modularity and cross-platform support.

Nginx and the C10K problem (2004)

Nginx used an event-driven architecture to handle many simultaneous connections with low memory usage.
